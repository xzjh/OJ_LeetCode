###Reverse words in a string

In Python, `split()` is different from `split(' ')`:

- `' '.split(' ')` => `['', '']`
- `' '.split()` => `[]`

###Evaluate reverse polish notation

In Python, `5/6` => `0` but `-5/6` => `-1`.
